Using the content property

We like to use ::before and ::after elements.
But these pseudo elements will not be generated if their content is set to 'none' (this is the default).
Use content: ""; with ::after and ::before pseudo-elements to get them to show up.
Or you can write a whole story. This one has content: 'To be or not to be, that is the question';
Neato! So that's all that content does, right?
You can use attr() to pull values from your html tags. This a tag has content: attr(href);
You can also combine attr() values and strings, like so: content: 'Visit this website!' attr(href);
You can also add your own custom tag attributes. This can be useful for custom tooltips. For example,
this tag's HTML looks like: <a href="#" tooltip="Don't forget to bring a towel!">
and has this CSS: content: attr(tooltip);
